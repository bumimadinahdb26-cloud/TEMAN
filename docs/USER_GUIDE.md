Panduan Pengguna TEMAN (Terapi dan Monitoring Akses Dialisis)
Selamat datang di aplikasi TEMAN. Dokumen ini akan membantu Anda memahami cara menggunakan aplikasi untuk monitoring kesehatan akses dialisis Anda.

1. Memulai Aplikasi
Saat pertama kali membuka aplikasi, Anda akan disambut dengan layar Login/Daftar. Untuk dapat menyimpan riwayat kesehatan Anda, Anda perlu membuat akun.

Mendaftar Akun Baru
Klik tab "Daftar".
Isi Nama Lengkap Anda.
Masukkan Email yang valid.
Buat Password (minimal 6 karakter).
Konfirmasi password Anda.
Pilih Avatar: Pilih salah satu dari 6 avatar yang tersedia atau upload foto profil Anda sendiri dengan menekan tombol +.
Klik tombol "Daftar".
Masuk (Login)
Jika Anda sudah memiliki akun:

Masukkan Email dan Password.
Klik "Masuk".
Anda juga bisa menggunakan tombol "Masuk dengan Google" (simulasi).
2. Mengatur Profil
Setelah login, Anda dapat mengatur profil Anda:

Klik avatar atau nama Anda di pojok kanan atas header.
Di jendela profil, Anda dapat:
Mengganti foto avatar (klik gambar avatar besar).
Mengubah nama tampilan.
Memilih Jenis Akses Dialisis yang Anda gunakan (CVC atau AV Shunt).
Klik "Simpan" untuk menyimpan perubahan.
3. Fitur Utama
A. Cek Risiko CVC
Fitur ini untuk pasien yang menggunakan kateter vena sentral.

Klik menu "CVC" atau kartu "Cek Risiko CVC".
Masukkan ID Pasien (opsional).
Centang gejala-gejala yang Anda rasakan:
Kemerahan
Bengkak
Nyeri
Nanah/Cairan (Tanda Kritis)
Demam (Tanda Kritis)
Klik "Hitung Risiko".
Hasil penilaian akan muncul (Aman, Rendah, Sedang, atau Tinggi).
Jika risiko tinggi/darurat, tombol untuk menghubungi layanan darurat akan muncul.
B. Cek Kondisi AV Shunt
Fitur ini untuk pasien dengan fistula atau graft.

Klik menu "AV SHUNT" atau kartu "Cek AV Shunt".
Masukkan ID Pasien (opsional).
Lakukan pemeriksaan mandiri:
Visual: Apakah ada kemerahan, bengkak, atau nanah?
Raba (Thrill): Apakah getaran terasa normal atau hilang?
Klik "Cek Kondisi".
Hasil akan menunjukkan status kondisi (BAIK, PERHATIAN, atau DARURAT).
C. Agenda & Acara
Melihat jadwal kegiatan edukasi atau acara rumah sakit.

Klik menu "AGENDA".
Lihat daftar acara mendatang.
Jika ada link pendaftaran, klik "Buka Link" untuk mendaftar.
D. Konsultasi
Menghubungi admin atau edukator.

Klik tombol "HUBUNGI ADMIN" di footer atau menu konsultasi.
Pilih admin yang ingin dihubungi.
Anda akan diarahkan ke WhatsApp.
4. Penyimpanan Data
Data Lokal: Data profil dan riwayat pengecekan Anda disimpan di Local Storage browser perangkat Anda. Data ini aman dan tidak hilang selama Anda tidak menghapus cache browser.
Avatar: Jika Anda mengunggah foto kustom, foto tersebut diubah menjadi format base64 dan disimpan bersama data profil.
Sinkronisasi: Fitur sinkronisasi cloud (Firebase) dapat diaktifkan dengan mengkonfigurasi js/firebase-config.js jika diperlukan pengembangan lebih lanjut.
5. Kebutuhan Sistem
Browser modern (Google Chrome, Mozilla Firefox, Safari, Edge).
Koneksi internet untuk memuat data agenda.
